# Quick Guide — Dr. Castro Website

## Run Locally

```bash
cd dr-castro-site
npm run dev
```

Open http://localhost:5173 in your browser.

---

## Build for Production

```bash
cd dr-castro-site
npm run build
```

Output in `dr-castro-site/dist/`.

---

## Tech Stack

- React + Vite
- Tailwind CSS v4
- Google Fonts (Cormorant Garamond, DM Sans)
